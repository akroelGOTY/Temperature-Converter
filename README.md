# Temperature-Converter

O **Temperature-Converter** é uma aplicação web que permite converter temperaturas entre diferentes escalas: Celsius, Fahrenheit e Kelvin. É uma ferramenta útil para estudantes, profissionais e qualquer pessoa que precise realizar conversões de temperatura de forma rápida e precisa.

## Linguagens Usadas

* **HTML**: Para a estrutura da página.
* **CSS**: Para o estilo e design da interface.
* **JavaScript**: Para a lógica de conversão e interação do usuário.

## Design e Cores

O design do Conversor de Temperatura é moderno e intuitivo, com uma paleta de cores que combina as seguintes tonalidades:

* **#68AEFF**: Um azul claro que representa frescor e clareza.
* **#003EB7**: Um azul escuro que traz profundidade e contraste.

A interface é responsiva, garantindo uma boa experiência em dispositivos móveis e desktops.

## Como Usar

1. Clone o repositório ou baixe o arquivo ZIP.
2. Abra o arquivo `index.html` em seu navegador.
3. Insira a temperatura que deseja converter.
4. Selecione a escala de origem (Celsius, Fahrenheit ou Kelvin).
5. Clique no botão "Converter" para ver os resultados.
